# back-express

Archetype with boilerplate code for a backend API with express

## Quick start

```bash
npm install -g --ignore-scripts=false @nubjs/nub   # one-time, system-level
nub node install 26 && nub node pin 26
nub install
nub run dev   # starts Express, serves API + static client
```
